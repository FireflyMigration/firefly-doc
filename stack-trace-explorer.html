Stack Trace Explorer

This Visual Studio extension allows quick navigation based on a call stack you receive from an error report by your customer. It automatically turns it into a set of hyper links that help you navigate the code.

You can download it from Visual Studio Marketplace

You can also find it's source on Github at: StackTraceExplorer on github

Thanks to Samir Boulema for writing this cool extension and being patient with my requests and issues
