Copy the Downloaded Files Copy

  1. Run the ENV.Web.Templates.vsix to install the visual studio templates used in this demo.
  2. Place the ENV.Web.dll in the bin directory of your Northwind Training folder.
  3. Open the WebProjects.zip and move the two folders in it to your Northwind Training folder.

  1. Right click on the Front End Project and select Open With Code

  1. You can change the Visual Studio Code color scheme to be like visual studio.

Important Note about video

In the video we used version 2017-09, please use version 2017-10
