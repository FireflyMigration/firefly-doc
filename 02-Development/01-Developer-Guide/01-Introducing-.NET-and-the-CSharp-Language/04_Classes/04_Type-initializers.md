Usage
#### without type initializers
```csdiff
-Animal c = new Animal();
-c.Name = "Snoopy";
-c.AnimalType = "Dog";
```

#### with type initializers
```csdiff
Animal c = new Animal
{
   Name = "Snoopy",
   AnimalType = "Dog"
};
```
