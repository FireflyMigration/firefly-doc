# Exercise - Using Class Outline to Read Large Controller Trees
1. Open one of your major classes.  
2. Remember to use (<kbd>Ctrl</kbd><kbd>m</kbd><kbd>o</kbd>) to collapse the code and improve readability.
3. Open the class out line panel.
4. Notice the tree like structure of the classes.
5. Right click one of the class, and navigate to the Run and Onload methods.
6. Find a UIControllers class and use class out line panel to navigate to the Form.
7. Use (<kbd>Ctrl</kbd><kbd>-</kbd>) to go back the class controller
