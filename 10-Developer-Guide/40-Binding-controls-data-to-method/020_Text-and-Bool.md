# Text and Bool Types

**Text**  
* Similar to - string
* Can be null
* When performing comparison it ignores spaces
    * For string : `"a " == "a"` return false
    * For Text : `"a " == "a"` return true
* Usage:
  * `Text a = ".NET";`

**Bool**  
* Similar to - bool
* Can be null as opposed to bool
* Usage:
  * `Bool a = true;`
  * `Bool b = null;`
