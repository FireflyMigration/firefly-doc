

* Similar to: int, decimal, float, double etc….
* Can be null as opposed to int, decimal etc….
* Can hold any int or decimal value
* When divided by Zero – returns Zero
* Usage:
    * `Number n = 1`;
    * `Number n2 = -1.5`;
