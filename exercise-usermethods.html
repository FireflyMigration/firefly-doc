Exercise - Input Validation using User Methods and the u Member

  1. In ShowProducts, add an error message if the ProductName has less than 3 characters (using the u.Len method). Use Trim where needed.
  2. Build and test.
  3. In Run Time use the Shift F12 key combination to go back to the code.
  4. Add the ProductName to the Watch screen.
  5. Add it again and add .Trim().
  6. Notice the different result.
  7. Add an error message in status bar if the ProductName contains one of the following characters: %, @.
  8. Build and test.
  9. Add a warning in status bar if the ProductName starts with "T" (using the u.Left method).
  10. Build and test.
