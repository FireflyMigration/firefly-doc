Operators Explanation Example
== Equals a == "London"
!= No Equals, different a != "London"
> Greater Than n > 5
< Less Than n < 7
<= Less or Equal n <= 6
>= Greater or Equal n >= 7
&& And n > 5 && a == "London"
|| Or n < 7 || a !="London"
! Not !(n < 7)
