Looking at a Large Migrated Program for the First Time exercise

  1. Choose a program
  2. Remember to use Ctrl + m, o to collapse the code.
  3. In it see that you can find and review the .NET equivilant of:
    1. Task Prefix
    2. Data Source
    3. Handlers
    4. Data View

Please write down the class you've chosen and the line numbers you've found it in.
