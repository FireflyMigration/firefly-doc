Exercise: Inheritance_Basics

  1. Under Exercises folder and under the MoreOnClass folder Create two new class :
    1. Main inheriting from Car.
    2. Weekend inheriting from Car.

Your new two class should look like :

using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;

namespace Northwind.Exercises.MoreOnClass
{
     class Main : Car
     {
     }
} 
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;

namespace Northwind.Exercises.MoreOnClass
{
     class Weekend : Car
     {
     }
}
  1. Add to the Main class, a new method name it DriveFast, the new method will show a message "Whoop I am fast".
  2. Add to the Weekend class, a new method name it DriveInStyle, the new method will show a message "I am cruising on the highway".

The two class should look like :

using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;
using System.Windows.Forms;

namespace Northwind.Exercises.MoreOnClass
{
    class Main : Car
    {
        public void DriveFast()
        {
            MessageBox.Show("Whoop I am fast");
        }
    }
}

using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;
using System.Windows.Forms;

        public void DriveInStyle()

}
  1. Add constructor to the two class.
  2. In the constructors initialize the carType. The two class should look like :
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;
using System.Windows.Forms;

namespace Northwind.Exercises.MoreOnClass
{
    class Main : Car
    {
         public Main()
         {
             carType = "Main";
         }
        public void DriveFast()
        {
            MessageBox.Show("Whoop I am fast");
        }
    }
} 
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;
using System.Windows.Forms;

namespace Northwind.Exercises.MoreOnClass

             carType = "Weekend";
         }

        {
            MessageBox.Show("I am cruising on the highway");
        }
    }
}
  1. In the TestClass change the first 2 cars declarations to use the right class.
  2. Remove the carType update from the first 2 cars declarations.
  3. Add a call to the cars DriveFast method and the DriveInStyle method.
  4. Notice that:
    1. mainCar do not have the DriveInStyle method.
    2. weekendCar do not have theDriveFast method.
    3. funCar do not have any of the above methods.
  5. Build and Test.

The TestClass class should look like :
