Visual Studio Extentions

The Firefly extensions bundler includes the following extensions, if the bundle installation failed please install them manually:

  1. Class Outline - A tool that shows you a window with all the classes and inner classes in a C# code file. When you right click on any node, you'll see the different important methods of it, such as OnStart etc... and will also be able to go directly to the controller's view
  2. VS Drop Assist - Allows for drag and drop of various objects from the solution into the code.
  3. Visual Studio Templates - Install item templates, snippets and help tune up Visual Studio.
